OVERVIEW FOR DEPLOYING WINDOWS UPDATES

The following procedures describe how to install the latest Windows OS versions. You can accomplish this by first creating a Group Policy Object (linked to an Organizational Unit), loading a WSUS administrative template, configuring automatic updates, and then specifying update service locations. Windows OS updates can be configured using Group Policy settings for Windows Server Update Services (WSUS). The Group Policy Object (GPO) will be linked to an Active Directory container appropriate for your environment. Configuring Windows OS updates also includes Internet Explorer browser updates. NOTE: We do not recommend editing the Default Domain or Default Domain Controller GPOs to add WSUS settings.
